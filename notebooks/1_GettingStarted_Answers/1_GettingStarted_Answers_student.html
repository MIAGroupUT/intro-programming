Exercise 1.5

Consider the code below. The programmer wanted to swap the values in the variable a and b. However, it does not work in the way the programmer expected.

  1. Explain why the program fails to swap the values.

  2. Fix the program so that the variables are swapped. Verify your program in Python.

a = 19
b = 7
a = b
b = a

Answer key 1.5

  1. Because the value of a is overwritten by that of b in line 3, the value of a gets lost before it can be used as the new value of b in line 4.

  2. Multiple options here. One is to use multiple assignments, i.e., b, a = a, b. The other one (which will more likely be used) is to add a temporary variable. E.g.,

a = 19
b = 7
temp = a 
a = b
b = temp

Exercise 1.6

Special numbers like \(\pi\) and Euler’s number \(e\) are not automatically defined in Python. Define these two numbers as variables and use them to compute the volume of a sphere with radius \(e\). Store the result in a new variable called v.

Paste the contents of your terminal window into your Word document.

Answer key 1.6

Students have to know how to compute the volume of a sphere, i.e., \(v = \frac{4}{3} \pi r^3\). They could define variables \(pi\) and \(e\) to achieve this

>>> pi = 3.14159265359
>>> e = 2.71828182845
>>> v = (4/3)*pi*(e**3)
>>> v
84.13410032088953

Exercise 1.7

Define two variables. One should have the value ‘Technical ‘ and the other ‘Medicine’. What is the type of these variables? Try to add these two variables in the Python interpreter. Which result do you get? Can you explain what happened?

Answer key 1.7

The type should be string or <class 'str'>. If you add these two, you get a new string ‘Technical Medicine’. Apparently, this is what the addition operator does for string variables.

Exercise 1.13

Write Python scripts for the following tasks. Use the input() and print() functions.

  1. A script that asks the user for the radius of a sphere and prints the volume of the sphere.

  2. A script that asks the user for two numbers and prints whether the first number is larger than the second number.

  3. A script that asks the user for numbers \(a\), \(b\) and \(c\) and prints whether \(a\mod b > c\).

  4. A script that asks the user for the height, width and depth of a box and returns it’s volume and surface area.

Answer key 1.13

Here, it’s important that you check if the scripts actually work with input that you provide. I’m not providing code examples here, but some pointers

  1. Students can reuse what they wrote earlier to compute the volume of the sphere.

  2. Here they can use comparison operators. They should also use casting here to change the input strings into floats or ints.

  3. Combination of modulo operator and comparison operators.

  4. Here three variables should be prompted. Surface area might be difficult for some.
